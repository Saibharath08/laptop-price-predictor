# Laptop Price Predictor

This project predicts laptop prices based on specifications using Machine Learning.

## Features
- Predict laptop price based on brand and specs
- User-friendly Streamlit interface
- Common-sense validations for realistic inputs

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit

## Input Features
- Brand
- Type
- RAM
- Weight
- Touchscreen
- IPS Display
- PPI
- CPU Brand
- HDD
- SSD
- GPU Brand
- Operating System

## How to Run
1. Clone repository
2. Install requirements:
pip install -r requirements.txt

3. Run app:
streamlit run app.py
